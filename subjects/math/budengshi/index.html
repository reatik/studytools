A. 2 个 B. 3 个 C. 4 个 D. 5 个
【答案】
C —— 不等式是用不等号（>、<、≥、≤、≠）连接的式子。①3<5，②4x+5>0，⑤x≠-4，⑥x+2≥x+1 都是不等式，共 4 个。
2. 设 a 是大于 1 的有理数，如果 a，(a+23)，(2a+13) 在数轴上对应的点分别为 A、B、C，那么这三个点在数轴上从左到右的顺序是（　　）
(A) C、B、A (B) B、C、A (C) A、B、C (D) C、A、B
【答案】
B —— 因为 a>1，所以 a+23 - a = a+2-3a3 = 2-2a3 < 0，即 a+23 < a。又 2a+13 - a+23 = a-13 > 0，即 2a+13 > a+23。所以 a+23 < 2a+13 < a，即 B、C、A。

【答案】
C —— 由数轴可知：-1<a<0，b>1。所以 a<0，b-1>0，因此 a(b-1)<0。
4. 若 a+b<0，且 -(-a)>0，则（　　）

5. 设 a、b、c 表示三种不同物体的质量，用天平称两次，情况如图所示，则这三种物体的质量从小到大排序正确的是（　　）

15.1(2) 不等式及其性质(2)

要点归纳

不等式性质3 不等式的两边同时加上（或减去）同一个数或同一个含有字母的式子，不等号的方向不变。即：

如果a>b，那么a+m>b+m；

如果a<b，那么a+m<b+m。

不等式性质4 不等式的两边同时乘以（或除以）同一个正数，不等号的方向不变。即：

如果a>b，且m>0，那么am>bm（或 am>bm）；

如果a<b，且m>0，那么am<bm（或 am<bm）。

不等式性质5 不等式的两边同时乘以（或除以）同一个负数，不等号的方向改变。即：

如果a>b，且m<0，那么am<bm（或 am<bm）；

如果a<b，且m<0，那么am>bm（或 am>bm）。

16. 比较两个代数式M、N的大小，只要作出差M-N。若M-N>0，则M>N；若M-N=0，则M=N；若M-N<0，则M<N。这种方法叫做"作差法"，利用该方法解决下面问题：
(1) 比较4+3a²-2b+b²与3a²-2b+1的大小；
(2) 若2a+2b-1>3a+b，比较a、b的大小；
(3) 已知A=2x²-1，B=x²-2x+1x-1，当x>-1时，比较A与1B的大小，并说明理由。
【答案】
(1) 4+3a²-2b+b² > 3a²-2b+1 —— 作差：(4+3a²-2b+b²)-(3a²-2b+1)=b²+3>0；
(2) b > a —— 移项得2a+2b-1-3a-b>0→-a+b-1>0→b-a>1>0→b>a；
(3) 当-1<x<1时，A>1B；当x=1时，B无意义；当x>1时，A<1B —— 1B=x-1(x-1)²=1x-1（x≠1），A-1B=2x²-1-1x-1=2-(x+1)(x+1)(x-1)=1-x(x+1)(x-1)=-1x+1。当x>-1时，x+1>0，所以A-1B<0，即A<1B（x≠1）。

15.2(1) 不等式的解和解集

要点归纳

1. 一元一次不等式的概念：只含有一个未知数，且含有未知数的式子都是整式，未知数的次数是1的不等式，叫做一元一次不等式。

2. 不等式的解：使不等式成立的未知数的值，叫做不等式的解。

3. 不等式的解集：一个含有未知数的不等式的所有的解，组成这个不等式的解集。

4. 求不等式解集的过程叫做解不等式。

例题

已知如图是关于x的不等式2x-a>-3的解集，则a的值为__________。

解：解不等式2x-a>-3得：x>a-32，

由数轴知不等式的解集为x>-1，

∴a-32=-1，

解得：a=1。

一、选择题
1. 下列不等式中，是一元一次不等式的是（　　）
A. 2x-1>0 B. -1<2 C. 3x-2y≤-1 D. y²+3>5
【答案】
A —— 一元一次不等式需满足：只含一个未知数，未知数次数为1，且是整式。A符合条件。
2. 不等式1+x≥-1的解集是（　　）
A. x≥2 B. x≥0 C. x≥-2 D. x≤0
【答案】
C —— 1+x≥-1 → x≥-2。
3. 下列不等式的解集中，不包括-5的是（　　）
A. x≤5 B. x≥-5 C. x≤-6 D. x≥-6
【答案】
C —— x≤-6表示所有小于等于-6的数，-5不在这个范围内。
4. 下列说法正确的是（　　）
A. y=4是不等式y+4<5的解 B. y=2是不等式2y<7的解集 C. 不等式2x<6的解集是x=3 D. y=2是不等式3y≥6的一个解
【答案】
D —— A错误（4+4=8不小于5）；B错误（y=2是一个解，不是解集）；C错误（解集是x<3）；D正确（3×2=6≥6）。
5. 不等式2(1-x)>x-6的解集在数轴上表示为（　　）
A. 数轴：-1到5，空心圆点在5，向左 B. 数轴：-1到5，空心圆点在5，向右 C. 数轴：空心圆点在83，向左 D. 数轴：空心圆点在83，向右
【答案】
C —— 2(1-x)>x-6 → 2-2x>x-6 → -3x>-8 → x<83。
6. 关于x的一元一次不等式的解集如图所示，且该不等式的负整数解有且只有四个，则m的取值范围是（　　）（数轴：空心圆点在m+1，向左）
A. -4≤m≤-3 B. -6≤m<-5 C. -5<m≤-4 D. -3<m<-2
【答案】
C —— 负整数解为-4,-3,-2,-1，共四个，所以m+1的范围是-4<m+1≤-3，即-5<m≤-4。
二、填空题
7. 若3-4x3-2n>2是一元一次不等式，则n=__________。
【答案】
1 —— 由3-2n=1得n=1。
8. 若(n-2)y|n|-1+29>0是关于y的一元一次不等式，则n的值为__________。
【答案】
-2 —— |n|-1=1且n-2≠0，所以|n|=2且n≠2，故n=-2。
9. 写出一个解集为x<5的一元一次不等式：__________。
【答案】
x+1<6（答案不唯一）
10. 关于x的方程x-5=-3a解为负数，则实数a的取值范围是__________。
【答案】
a>53 —— x=5-3a<0 → 5-3a<0 → a>53。
11. 下图表示的关于x的不等式的解集是__________。（数轴：空心圆点在-3，向右）
【答案】
x>-3
12. 若关于x的不等式2x-a>-3的解集如图所示（数轴：空心圆点在-1，向右），则a的值为__________。
【答案】
1 —— 2x-a>-3 → x>a-32，由数轴知a-32=-1 → a=1。
13. 若关于x的不等式x-m≥0的最小整数解是x=2，则m的取值范围是__________。
【答案】
1<m≤2 —— x≥m，最小整数解为2，则1<m≤2。
14. 若关于x的方程x-23=m+12的解是负数，则m的取值范围是__________。
【答案】
m<-53 —— 2(x-2)=3(m+1) → 2x-4=3m+3 → 2x=3m+7 → x=3m+72<0 → 3m+7<0 → m<-73。
15. 如果不等式2x<6的解都能使不等式x-a<5成立，那么a的取值范围是__________。
【答案】
a≥-2 —— 2x<6→x<3；x-a<5→x<a+5。要使所有x<3都满足x<a+5，则a+5≥3→a≥-2。
三、解答题
16. 解不等式：x+23≥x-2，并写出所有符合条件的正整数解。
【答案】
解：x+2≥3(x-2) → x+2≥3x-6 → -2x≥-8 → x≤4。正整数解为1, 2, 3, 4。
17. 已知关于x的方程x+12=1-x-15的解也是不等式2x-3a<5的一个解，求满足条件的整数a的最小值。
【答案】
解方程：5(x+1)=10-2(x-1) → 5x+5=10-2x+2 → 7x=7 → x=1。代入不等式：2×1-3a<5 → -3a<3 → a>-1。整数a的最小值为0。
18. 已知关于x的两个不等式：①2(x-1)<a+5；②2x<4。
(1) 若两个不等式的解集相同，求a的值；
(2) 若不等式①的解都是不等式②的解，求a的取值范围。
【答案】
(1) ①2x-2<a+5→2x<a+7→x<a+72；②x<2。解集相同则a+72=2→a+7=4→a=-3；
(2) 要使①的解都是②的解，则a+72≤2→a+7≤4→a≤-3。
19. 定义运算：a*b=a-2b，例如：1*2=1-2×2=-3。若不等式x*a<1的解集在数轴上如图所示（数轴：空心圆点在1，向左），求a的值。
【答案】
x*a=x-2a<1→x<2a+1。由数轴知解集为x<1，所以2a+1=1→a=0。
20. 我们把关于x的一个一元一次方程和一个一元一次不等式组成一种特殊组合，当一元一次方程的解正好也是一元一次不等式的解时，我们把这种组合叫做"有缘组合"；当一元一次方程的解不是一元一次不等式的解时，叫做"无缘组合"。
(1) 组合2x-4=05x-2<3是__________（填有缘组合或无缘组合）；
(2) 若关于x的组合5x+15=03x-a2>a是"有缘组合"，求a的取值范围。
【答案】
(1) 解方程2x-4=0得x=2；解不等式5x-2<3得x<1。x=2不是x<1的解，所以是无缘组合；
(2) 解方程5x+15=0得x=-3。代入不等式：3×(-3)-a2>a→-9-a2>a→-9-a>2a→-9>3a→a<-3。
