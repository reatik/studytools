一、选择题
1. 式子：①3<5；②4x+5>0；③x=3；④x²+x；⑤x≠-4；⑥x+2≥x+1．其中是不等式的有（　　）
A. 2 个 B. 3 个 C. 4 个 D. 5 个
【答案】
C —— 不等式是用不等号（>、<、≥、≤、≠）连接的式子。①3<5，②4x+5>0，⑤x≠-4，⑥x+2≥x+1 都是不等式，共 4 个。
2. 设 a 是大于 1 的有理数，如果 a，(a+2)/3，(2a+1)/3 在数轴上对应的点分别为 A、B、C，那么这三个点在数轴上从左到右的顺序是（　　）
(A) C、B、A (B) B、C、A (C) A、B、C (D) C、A、B
【答案】
B —— 因为 a>1，所以 (a+2)/3 - a = (a+2-3a)/3 = (2-2a)/3 < 0，即 (a+2)/3 < a。又 (2a+1)/3 - (a+2)/3 = (a-1)/3 > 0，即 (2a+1)/3 > (a+2)/3。所以 (a+2)/3 < (2a+1)/3 < a，即 B、C、A。
3. 如图，下列正确的是（　　）
A. a/b > 0 B. a-b > 0 C. a(b-1) < 0 D. (a+1)(b-1) < 0
【答案】
C —— 由数轴可知：-1<a<0，b>1。所以 a<0，b-1>0，因此 a(b-1)<0。
4. 若 a+b<0，且 -(-a)>0，则（　　）
A. a>0, b<0 B. a>0, b>0 C. a<0, b>0 D. a<0, b<0
【答案】
A —— 因为 -(-a)=a>0，所以 a>0。又因为 a+b<0 且 a>0，所以 b<-a<0，即 b<0。
5. 设 a、b、c 表示三种不同物体的质量，用天平称两次，情况如图所示，则这三种物体的质量从小到大排序正确的是（　　）
A. c<b<a B. b<a<c C. c<a<b D. a<b<c
【答案】
A —— 由第一个天平可知：b>c。由第二个天平可知：b<a。所以 c<b<a。
二、填空题
6. 有下列式子：①-2>0；②4x+y≤1；③x+3=0；④y-7；⑤m-2.5>3．其中是不等式的有______个．
【答案】
3 —— 不等式是用不等号连接的式子。①-2>0，②4x+y≤1，⑤m-2.5>3 都是不等式，共 3 个。
